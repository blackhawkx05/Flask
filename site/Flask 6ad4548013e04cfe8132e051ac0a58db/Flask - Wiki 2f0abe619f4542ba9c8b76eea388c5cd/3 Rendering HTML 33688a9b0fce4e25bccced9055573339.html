3. Rendering HTML

Import all required functions

from flask import Flask, render_template

storrage place: in a templates folder

a example html:

<!doctype html>
<html>
    <head>
        <title>Home page</title>
    </head>
    <body>
        <h1>Home Page!</h1>
    </body>
</html>

render the html file in pythn: the name of the main html file must be called = index.html

@app.route("/")
def home():
    return render_template("index.html")

Dynamic Html

html example: content is a variable

<!doctype html>
<html>
<head>
    <title>Home page</title>
    </head>
    <body>
	<h1>{{content}}</h1>
    </body>
</html>

pythn function: content is the variable which is shown in the html

@app.route("/")
def home():
    return render_template("index.html", content="Testing")

run python code in html:

<!doctype html>
<html>
<head>
    <title>Home page</title>
    </head>
    <body>
	{% if content == "true" %}
            <p>True!</p>
        {% else %}
            <p>False :(</p>
        {% endif %}
    </body>
</html>
<!doctype html>
<html>
<head>
    <title>Home page</title>
    </head>
    <body>
	{% for x in range(10) %}
             <p>{{x}}</p>
        {% endfor %}
    </body>
</html>

multiple balues:
